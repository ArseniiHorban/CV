Event Booking Platform

A web application for booking. Bookings are automatically synchronized from Google Calendar, making it easier to manage and edit events (e.g., time or date) without the need for a its own advanced editing interface. The booking flow is optimized for usability and includes server-side control over available time slots based on current calendar data.

Django Celery Redis Tailwindcss Google Calendar API HTML CSS

My role:

Backend development: implementing logic for bookings, validating time-based constraints, and integrating Google Calendar sync. Also responsible for asynchronous task handling using Celery and Redis, and managing calendar conflict resolution.

Challenges overcome:

Implemented Google Calendar API sync and maintained stable background communication with external services. Designed logic to prevent double bookings, time collisions and databases conflicts. Calculated free time windows dynamically, taking into account event duration and required cooldown periods between sessions.

Crypto Tracker

A web platform that displays real-time cryptocurrency market data, charts, and analytics. Users can register and manage their personal crypto portfolios by specifying the amount of each asset they hold. The system calculates the portfolio’s total value based on live prices and visualizes the data using pie charts and dynamic metrics

Django Redis Google Oauth2 reCAPTCHA JavaScript Chart.js Docker HTML CSS

My role:

Full-stack development: building user authentication and profile systems including designing and managing database, implementing external APIs integration and caching, creating real-time data visualizations, and designing the logic of the app.

Challenges overcome:

Implemented reliable API data fetching with caching to improve performance. Developed a full user authentication system supporting both Google OAuth and traditional login/password with CAPTCHA protection. Designed and built secure data storage and dynamic computation for individual user portfolios, along with interactive chart visualizations and graph portfolio metrics.
